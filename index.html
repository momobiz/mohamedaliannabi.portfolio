My projects

March, 2023

Framework testing website using a BDD approach

In this demo project, I came back to the tutorial website dedicated to automation of test cases and API : AutomationExercise. The website is an e-commerce shop exposing different clothes for different type of clients such as male, female and kids. Each product has a name, a brand, a price and a category. The big advantage of the website is that it gives a help for automation test learner a detailed description of 26 test case and some informations about 13 api.

I designed and realized a first framework implementing 18 among 26 of the proposed test cases. My framework has the following caracteristics :
   ⏩ Application of the BDD approach
   ⏩ Usage of Selenium WebDriver Java library
   ⏩ Application of Page Object Model and Page Factory
   ⏩ Reporting using Extent Report
   ⏩ Storing all the user's data in Excel sheet
   ⏩ Running build on Jenkins and generating report
